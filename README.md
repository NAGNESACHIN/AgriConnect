# 🌱 AgriConnect

AgriConnect is a smart agriculture marketplace platform that connects farmers directly with buyers.  
The platform helps farmers upload and sell crops online while buyers can explore fresh farm products easily.

---

# 🚀 Features

## 👨‍🌾 Farmer Dashboard
- Upload crops with images
- Add crop price and description
- Manage uploaded products
- Delete products
- View earnings and orders

## 🛒 Buyer Dashboard
- Browse fresh crops
- Explore marketplace
- Buy directly from farmers

## 🔐 Authentication
- Login system
- Signup system
- Farmer & Buyer role selection
- LocalStorage based authentication

## 🌐 Marketplace
- Dynamic crop listings
- Real-time updates from farmer uploads
- Product cards with images

## 🎨 UI Features
- Responsive Design
- Modern Agriculture Theme
- Creative Branding & Logo
- Interactive Components

# ⚡ How To Run Project

## Frontend

Open:

```plaintext
index.html
```

in your browser.

---

## Backend Setup

Open terminal inside Backend folder:

```bash
cd Backend
npm install
node server.js
```

Server will run on:

```plaintext
http://localhost:3000
```
