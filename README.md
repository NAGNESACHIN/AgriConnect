# AgriConnect

A simple agriculture marketplace  project.

## Structure

- `index.html` — Landing page
- `assets/css/style.css` — Global styles
- `assets/js/main.js` — Basic page scripts
- `assets/js/auth.js` — Login/signup stubs
- `pages/` — App pages: login, signup, marketplace, buyer/farmer dashboards
- `Backend/server.js` — Minimal Express server

## Usage

Open `index.html` in a browser to view the landing page.

For backend support:

```bash
cd Backend
npm install
node server.js
```
